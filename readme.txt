Plugin for showing AppStore Apps Information

== Description ==
Use this Plugin if you want to display Itunes Apps information. This plugin updates all Data directly from the Apple Server.

You can easily add information from the Apple AppStore with using the 
shortcode [appstore id="idnumber"] in your posts, or pages. 
The ID number is the number from the official AppStore URL.

* [appstore id="idnumber"] with combined content tags gives you the data.

* Shortcode Content Tags: %description%, %name%, %version%, %genre%, %size%, %seller%, %releasenotes%, %released% , %appstoreurl% , %sellerurl% , %thumbnail% , %developerurl%, %numratings%, %avgrating%, %price%, %screenshots%

* example: [appstore id="284882215"]%name% %description% %thumbnail%[/appstore]

== Installation ==
1. Upload the `appster` Folder and its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View the Settings Page to view the available Appstore variables
